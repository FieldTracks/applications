Welcome to Fieldtracks!

Initial Setup

You have booted this Fieldtracks-instance for the very first time. In order to get system up and running, you need to set a few properties.

The properties are used for the both the server and ESP32-devices, also known as stones.

The Public Hostname is the DNS-Name of your server. It has to be set according to your TLS-certificate. It used by the user-interface and stones for establishing connections.

The Wireless Network configuration is transferred to stones when flashing the firmware. This is the on-site wireless network.

You can set Bluetooth parameters in the BLE Beacons section. Typically, you can leave it untouched. The first stone installation defaults to the stone's start-number. Smartphone beacons start at the smartphone's start number respectively. Beacon numbers are unique and sequential. When utilizing other beacons than stones (e.g. smartphones), reserving a dedicated block for stones (e.g. start = 512) is a good idea. Change this, if you already have beacons utilizing this Network. The namespace is derived from the public hostname (first 10 byte).
